# Path Branch Camera

Path Branch Camera is a web-based camera application built with React that allows users to access their device camera, capture images, and process them through different decision-based paths.

After capturing an image, the application can perform different actions such as previewing, saving, or preparing the image for further processing with external tools like AI models (e.g., Ollama).

This project is a learning experiment to combine browser camera access, image handling, and simple branching logic with future AI integration possibilities.

---

## Features

- Access device camera in browser
- Capture images using webcam
- Preview captured photos
- Simple branching logic for different actions
- Ready for AI integration (Ollama or APIs)
